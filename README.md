##  Learning Points

###  Spring Framework
- Spring MVC의 핵심 동작 원리(DispatcherServlet, Filter, Interceptor) 파악
- 검증(Validation), 예외 처리(Exception Handling) 등 견고한 웹 애플리케이션 구축
- 쿠키와 세션을 활용한 로그인 처리 및 보안 기초 학습
- 파일 업로드/다운로드 기능 구현

###  Algorithm & Coding Test
- 문제의 핵심을 파악하고 적절한 자료구조(Stack, Queue, Graph 등) 선택 능력 배양
- 시간 복잡도를 고려한 효율적인 알고리즘 설계 (DFS/BFS, Greedy, DP)

###  Database (SQL)
- RDBMS의 기본 이론 이해 및 정규화
- `JOIN`, `GROUP BY`, `Subquery` 등을 활용한 복잡한 데이터 집계 및 분석
